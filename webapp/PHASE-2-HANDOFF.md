# CleanLA Phase 2 Handoff

## What Changed

- Added the Phase 2 spots schema migration with enums, `spots`, `spot_media`, spatial index, public read RLS, legacy `reports` migration, and `spots_in_bounds`.
- Added seeded Los Angeles spots across categories and statuses.
- Added `GET /api/spots` with numeric bounds validation and a capped default limit of 500.
- Replaced the home page with a client Mapbox map shell, clustered category pins, custom 369 zoom controls, status strip, missing-token state, and spot detail sheet.
- Added PWA manifest basics and a square CleanLA SVG icon.
- Installed `react-map-gl` and `mapbox-gl`.

## Verification

- `npm run lint` passed.
- `npm run typecheck` passed.
- `npm run build` passed.
- `GET /api/spots` invalid bounds test returned `400`.
- `/manifest.webmanifest` is generated by Next.
